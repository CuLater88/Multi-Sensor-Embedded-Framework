# Multi-Sensor-Embedded-Framework
This Arduino-based monitoring system collects and displays real-time data from multiple sensors, including a capacitive soil moisture sensor, temperature sensor, photoresistor, and ultrasonic distance sensor. The project demonstrates the integration of both analog and digital peripherals within a microcontroller environment.

The primary goal of this project was to develop a stronger foundation in C++ by implementing object-oriented design principles such as classes, encapsulation, and modular code organization. Additionally, the project provided valuable hands-on experience with sensor interfacing, signal acquisition, and the practical application of analog and digital systems in embedded engineering.
